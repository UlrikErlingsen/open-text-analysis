# Contributing

TextSignal welcomes reproducible bug reports, documentation improvements, analytical fixtures, and narrow pull requests.

1. Explain the corpus shape and expected behavior without sharing private responses.
2. Add deterministic tests for analytical changes.
3. Run `pytest`, `ruff check .`, and `python -m build`.
4. Keep raw text, context snippets, IDs, and row assignments out of exported evidence.
5. Preserve bounded claims: lexical components are not validated themes and stability is not validity.
6. Cite public methodological sources and do not add proprietary course content or branding.
